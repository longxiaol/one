gulp
	安装 nodeJS
	安装 cnpm
		npm install -g cnpm --registry=https://registry.npm.taobao.org
	全局 gulp 安装
		cnpm install gulp -g
	使用 npm init 在项目目录中生成 package.json 文件

	package.json文件作用：
		保存项目配置信息，存放项目使用到的各种插件名称、版本信息，以便于项目在拷贝时，不需要拷贝 node_modules 文件夹下内容

	本地安装 gulp：
		在项目目录中使用 cnpm install gulp --save-dev 本地安装
	安装 gulp 插件
		gulp-clean-css: cnpm install gulp-clean-css --save-dev
			作用：压缩CSS
		gulp-uglify: 
			作用：压缩JS
		gulp-sass:
			作用：编译SASS（将.scss文件编译为 .css文件）
		gulp-livereload:
			作用：自动刷新


	http-server:
		0配置服务器，可以把任意的目录变成服务器目录
	全局安装 http-server 服务器
		cnpm install http-server -g
	在需要变成服务器的目录命令提示符下执行 http-server 命令
